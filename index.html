Introdução - O que é o Razor?


Razor é uma sintaxe view engine do ASP.NET usado para criação de páginas dinâmicas com C#, .NET ou Visual Basic.

Seu desenvolvimento teve inicio em Junho de 2010, sendo lançado pela Microsoft em Janeiro de 2011 junto com o ASP.NET MVC3 no Visual Studio 2010 e Microsoft WebMatrix.

Baseado na sintaxe do C#, Razor permite o desenvolvedor usar um workflow parecido com HTML, ao invés dos blocos do ASPX <% %>

Sua sintaxe sempre começa com o @ e nem sempre é necessário fechar o bloco de código

A ideia do Razor otimizar o workflow do desenvolvedor, reduzindo a quantidade de keystrokes (teclas pressionadas) e facilitando o entendimento do código, pois se comparado com a sintaxe ASPX, Razor é muito mais semântico.

Vantagens do Razor

Não é uma linguagem nova

Compacta

Workflow fluído

Fácil de aprender e usar

Compatível com testes unitários

Desvantagens do Razor

O código pode ficar um pouco confusso

Fácil confundir tags Server side com Client side

Helpers

O Razor conta com alguns Helpers para facilitar a vida do desenvolvedor:

Chart: Processa um gráfico, oferecendo os mesmos recursos como o controle de gráfico no ASP.NET 4;

Webgrid: Processa uma grade de dados, com funcionalidade de paginação e ordenação;

Crypto: Utiliza algoritmos de hash para criar senhas devidamente criptografadas;

WebImage: Processa uma imagem;

WebEmail: Envia uma mensagem de e-mail;

Conversões

O Razor possuí alguns metódos que auxiliam na conversão dos tipos de dados:

IsInt: Verifica se é um tipo inteiro válido;

AsInt: Converte uma string para inteiro; 


var myIntNumber = 0;   
var myStringNum = "539";   
if(myStringNum.IsInt() == true) {  
   myIntNumber = myStringNum.AsInt();   
}

IsDateTime: Verifica se é um tipo data-hora válido;

AsDateTime: Converte uma string (ex.: "31/12//2069") para data-hora; 


var myDateString = "31/12//2069";
var newDate = myDateString.AsDateTime();

// Retorna DateTime.MinValue ou o valor especificado, 
// se o valor da string é inválido

IsBool: Verifica se é um tipo booleano válido;

AsBool: Converte uma string "true"/"false" para booleano; 


var myStringBool = "True";
var myVar = myStringBool.AsBool();

// Retorna false ou o valor especificado,
// se o valor da string é inválido

IsFloat: Verifica se é um tipo float válido;

AsFloat: Converte uma string que tem um valor decimal (ex.: "6.66") para float; 


var myStringDec = "6.660";
var myDecNum = myStringDec.AsFloat();

// Retorna "0.0" ou o valor especificado,
// se o valor da string é inválido

IsDecimal: Verifica se é um tipo decimal válido;

AsDecimal: Converte uma string que tem um valor decimal (ex.: "6.66") para decimal (No ASP.NET, decimal é mais preciso que float); 


var myStringDec = "6.660";
var myDecNum = myStringDec.AsDecimal();

// Retorna "0.0" ou o valor especificado,
// se o valor da string é inválido

ToString: Converte qualquer tipo para uma string; 


int num1 = 17;
int num2 = 76;

string myString = num1.ToString() + num2.ToString();

// 1776
